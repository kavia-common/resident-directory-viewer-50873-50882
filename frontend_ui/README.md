# Resident Directory - React SPA

This frontend-only app displays a list of residents with search and a responsive detail view. No backend is required; data is loaded from a local static JSON file.

## Run locally

- npm install
- npm start
- Open http://localhost:3000

## Features

- Search by name (debounced)
- Selected resident retained in URL (?residentId=ID)
- Responsive detail panel (side panel on desktop, modal on small screens)
- Accessible UI with light theme and primary #3b82f6 / success #06b6d4 accents
